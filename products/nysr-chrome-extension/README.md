# NYSR AI Companion — Chrome Extension

AI-powered content assistant for NY Spotlight Report operations.

## Features
- **Compose**: Generate LinkedIn posts, emails, tweets, blog content with one click
- **Analyze**: Summarize any page, extract business intelligence, identify leads
- **Quick Actions**: Pre-built templates for all common content types
- **Context Menu**: Right-click any page to generate content from selection

## Installation (Development)
1. Clone/download this folder
2. Open Chrome → navigate to `chrome://extensions/`
3. Enable "Developer mode" (top right toggle)
4. Click "Load unpacked"
5. Select this `nysr-companion` folder
6. Extension icon appears in toolbar

## Setup
1. Click the extension icon
2. Go to Settings tab
3. Enter your Anthropic API key (from console.anthropic.com)
4. Save — the status dot turns green

## Publishing to Chrome Web Store
1. Zip the entire folder: `zip -r nysr-companion.zip nysr-companion/`
2. Visit: https://chrome.google.com/webstore/devconsole
3. Click "New Item" → upload ZIP
4. Fill in store listing details
5. Submit for review (2-3 business days)
6. Cost: $5 one-time developer fee
