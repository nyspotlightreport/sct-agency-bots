LinkedIn Access Token — 5 Minutes

You have the LinkedIn app (Client ID + Secret). The missing piece is an access token from the OAuth flow. Here's the fastest path.

1
Go to LinkedIn Developer Portal → your app → Auth tab
Open LinkedIn Apps →
2
Under "OAuth 2.0 tools" → click Request access token
Scopes needed: w_member_social + r_liteprofile
3
Copy the generated token → add as GitHub secret LINKEDIN_ACCESS_TOKEN
Add Secret →
4
LinkedIn tokens last 60 days. Add a calendar reminder to refresh it every 50 days.
⚡ After adding the token — LinkedIn bot posts once daily: insights post (~180 words) with link to blog. Target: 3,000+ impressions per post at 500 connections.
