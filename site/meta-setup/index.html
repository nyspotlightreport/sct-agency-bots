📘 Meta / Facebook App Setup

The Meta app is already created (S.C. Thomas). We need to get the App ID, App Secret, and Page Token.

Step 1 — Get App ID and Secret
Go to your Meta for Developers app:

Open My Meta Apps →

Click your app → Settings → Basic
Copy the App ID and App Secret
Step 2 — Get Page Access Token
In your Meta app → Tools → Graph API Explorer
Select your Page from the dropdown → Generate Token

Permissions needed: pages_manage_posts, pages_read_engagement, publish_to_groups, instagram_basic, instagram_content_publish
