Etsy Auto-Lister Setup (one-time, 8 min)

Once done, bot creates all 10 listings automatically. 92M buyers.

Step 1 — Create Etsy Developer App

Go to Etsy Developers

App name: "NYSR Store Bot" · Callback URL: https://nyspotlightreport.com

Copy your API Key (Keystring) and Shop ID (from etsy.com/shop/YourShopName → Settings → Info & Appearance → scroll to Shop ID)

Step 2 — Add to GitHub Secrets

Add ETSY_API_KEY Add ETSY_SHOP_ID

Step 3 — Run the Bot (1 click)

Run Etsy Auto-Lister

Click "Run workflow" → Done. All 10 listings created as drafts.

Then upload PDF files in Etsy dashboard and publish each one.

After setup: Etsy bot runs automatically. 10 products on 92M-buyer marketplace. $0 monthly cost.
