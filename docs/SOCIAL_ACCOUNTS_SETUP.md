## Free / Low-Cost Scheduling Tools

| Tool       | Price     | Best For                         | URL                        |
|------------|-----------|----------------------------------|----------------------------|
| Buffer     | $15/mo    | Simple scheduling, analytics     | https://buffer.com         |
| Later      | $18/mo    | Visual planning, Instagram-first | https://later.com          |
| TweetDeck  | Free      | Twitter/X only                   | https://tweetdeck.twitter.com |
| YouTube Studio | Free  | YouTube scheduling built-in      | https://studio.youtube.com |

**Recommendation**: Start with Buffer ($15/mo) for cross-platform scheduling. It covers Instagram, TikTok, Twitter/X, LinkedIn, and YouTube in one dashboard.
