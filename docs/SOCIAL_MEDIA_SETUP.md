# Social Media Account Setup Guide

## Overview
This guide outlines the 3-tier account structure for the SCT Agency social media empire.
Each tier serves a different strategic purpose in the content ecosystem.

---

These are the primary revenue-driving and authority-building accounts.

### 1. NY Spotlight Report (@NYSpotlightReport)

- **Content Source**: Original articles, investigative features, event coverage

## Account Creation Checklist

For each account:
- [ ] Create email address (use the emails listed above)
- [ ] Register account on each platform listed
- [ ] Set profile photo (brand logo or headshot for S.C. Thomas)
- [ ] Set banner/cover image (branded)
- [ ] Write bio using the text above
- [ ] Add website link
- [ ] Follow 50-100 relevant accounts in the niche
- [ ] Enable notifications for engagement
- [ ] Connect to social management tool (Buffer, Hootsuite, or custom)
- [ ] Test first post

## Cross-Promotion Strategy

- Tier 1 accounts promote each other regularly
- Tier 2 accounts always link back to Tier 1 content
- Tier 3 accounts drive directly to ProFlow landing pages
- All accounts use the Social Commander bot for automated content generation
- Content packages are generated daily by `agents/social_media/social_commander.py`

## Automation Notes

- Content is auto-generated by the Social Commander daily at 7am ET
- Packages are saved to `data/social/content_packages/` and `data/social/proflow_content/`
- Each package contains platform-specific formatting for all 8 platforms
- Manual review recommended before posting (especially Reddit and LinkedIn)
- ProFlow case study content is generated separately for maximum impact
